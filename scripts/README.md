# Scripts

One-off and utility scripts. Run from the project root unless noted.

| Script | Purpose |
|--------|---------|
| `apply-rls-fix.cjs` | Apply RLS fix (Supabase) |
| `check-db-data.cjs` | Check database data |
| `create-admin.cjs` | Create admin user |
| `debug-dashboard.cjs` | Debug dashboard |
| `fix-css.cjs` | CSS fix utility |
| `test-subscribers.cjs` | Test subscribers flow |
| `setup-crew-analysis-table.js` | Setup crew analysis table |
| `update-stripe-config.js` | Update Stripe config |
| `setup-env.js` | Environment setup helper |
| `deploy-check.sh` | Pre-deploy checks |
| `setup-api-keys.sh` | API keys setup |
| `setup-email-notifications.sh` | Email notifications setup |
| `test_image_processing.py` | Test image processing (Python) |

Use with `node scripts/<name>` or `bash scripts/<name>.sh` as appropriate. These are not part of the main app build.
